    Технологии

    В этом разделе описаны технологии и инструменты, с которыми я работаю уже долгое время.

    SAP Ecommerce - платформа, позволяющая предприятиям создавать и управлять интернет-магазинами для продажи своих товаров и услуг клиентам.

    Apache JMeter - популярный инструмент тестирования производительности с открытым исходным кодом, используемый для тестирования производительности и масштабируемости приложений.

    Postman - инструмент для разработки и тестирования API, позволяющий разработчикам легко создавать и управлять запросами API, просматривать ответы, а также обмениваться коллекциями

    Selenide - это небольшая и удобная библиотека Java для написания и выполнения автоматизированных тестов для веб-приложений, предоставляющая высокоуровневый API для написания тестов и скрывающая большую часть базовой сложности веб-тестирования.

    Dynatrace - программная интеллектуальная платформа, которая помогает организациям контролировать, анализировать и оптимизировать производительность своих цифровых экосистем, включая облачные нативные приложения, мультиоблачные инфраструктуры.

    Grafana - платформа визуализации данных и аналитики, позволяющая пользователям создавать, исследовать и совместно использовать интерактивные панели и графики реального времени для визуализации и анализа данных временных рядов из различных источников.
